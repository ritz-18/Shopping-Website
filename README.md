# Shopping Website

A simple and responsive Shopping Website built using HTML, CSS, and JavaScript.
This project provides a modern e-commerce interface where users can browse products, 
view details, add items to the cart, and enjoy a smooth shopping experience.

# Features

- Responsive design for mobile, tablet, and desktop
- Product Searching option 
- Product listing section
- Product details display
- Add to Cart functionality
- Cart item counter
- User-friendly interface
- Interactive buttons and animations
- Clean and organized code structure

# Technologies Used

- HTML5 : Structure of the website
- CSS3 : Styling and responsive design
- JavaScript (ES6) : Dynamic functionality and user interactions
